# Lunara
Interpreted programming language made using C++.

# Whole new programming language!
This programming language is still on development.
For now, you can do things like:
```
assign x = 34
assign y = 12
x + y
```
But soon, it will be more flexible. I will add compiler support, \
and I'll try to connect this language with python.
# What is this language for?
Its use will be general, but more oriented to maths right now, \
since it only covers mathematical expressions for now.

# Did I use any libraries?
No. I just wrote the parser, the lexer, the abstract syntax tree, and the main file from scratch. \
Since this task is hard, I started helping me with AI, but, I'm leaving it since I won't learn anything if i don't do it by myself.
